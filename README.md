# ICT Industry Tracking Platform

Static GitHub Pages version of the TEEMA ICT industry tracking prototype.

## Files

- `index.html` - current app entry, layout, and tracking data
- `support.js` - runtime used by the current app
- `foxconn-map.html` - interactive company site map
- `app.js` and `styles.css` - legacy static build files retained for reference

## GitHub Pages

Recommended repository name: `ict-industry`

Expected URL after Pages is enabled from the `main` branch root:
